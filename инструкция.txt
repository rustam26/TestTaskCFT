Команда для запуска: java -jar sort-it-1.0.jar
1. тип данных (-s или -i), обязательный;
2. режим сортировки (-a или -d), обязательный;
3. имя выходного файла, обязательное;
4. остальные параметры – имена входных файлов, не менее одного.
Примеры запуска из командной строки для Windows:
java -jar sort-it-1.0.jar -i -a out.txt in.txt (для целых чисел по возрастанию)
java -jar sort-it-1.0.jar -s out.txt in1.txt in2.txt in3.txt (для строк по возрастанию)
java -jar sort-it-1.0.jar -s -d out.txt in1.txt in2.txt (для строк по убыванию)
